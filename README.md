## 2019阅读计划(包含后台)

### 简介

创建自己的2019阅读计划

项目首页：

[预览链接](https://activity.dookbook.com/booklist/index.html)

### 技术点

* Vuejs
  * Vue-Router
  * Vuex
  * Axios
* 第三方
  * Element-UI
  * html2canvas：
    * html绘制图片
    * canvas跨域问题解决方案(踩坑)
  * vue-qr：
    * 生成二维码
* 微信jssdk：
  * 微信网页授权
  * 自定义分享(踩坑)
  * 通过微信调用原生功能
